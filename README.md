# aws-s3-sapi
MuleSoft AWS S3 System API

This AWS S3 System API provides Operations:
1. List all Objects in a given S3 Bucket
2. Get Object from S3 using Bucket Name and ObjectKey
3. Get Object Attributes from S3 using Bucket Name and ObjectKey
4. Create a new object in given S3 Bucket
5. Delete an Object S3 using Bucket Name and ObjectKey
